# 🚀 AI Stock Alert System

An AI-powered stock alert system that sends real-time alerts based on RSI and price triggers using automation workflows.

---

## 🔥 Features

* 📊 Real-time stock monitoring
* 📉 RSI-based buy/sell alerts
* 🔔 Telegram/WhatsApp notifications
* ⚡ Fully automated workflow

---

## 🛠️ Tech Stack

* Python / JavaScript
* n8n Automation
* Stock APIs (Yahoo Finance / NSE / BSE)

## 🚀 How It Works

1. Fetch stock data via API
2. Calculate RSI
3. Check conditions (overbought/oversold)
4. Trigger alerts automatically
